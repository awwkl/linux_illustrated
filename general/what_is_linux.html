What is Linux?

This article is still a stub :(

"Everything in Linux is a file"

is probably the most important thing to keep in mind while continuing to read this article. Everything is a file, from the bootloader executable file, to the compressed kernel image, the files in the filesystem hierarchy, and the system manager and user software binaries, libraries, and configuration files.

Overview - 4 main components

  1. Bootloader
  2. Kernel
  3. Filesystem hierarchy (directory structure and files)
  4. System manager and User packages
